# Google-Gemini-AI-Chatbot
AI-powered chatbot built using Google Gemini API that generates intelligent and context-aware responses to user queries. Developed with HTML, CSS, and JavaScript, featuring a clean UI and real-time interaction. Demonstrates API integration, asynchronous JavaScript, and dynamic content rendering.
This project is an AI-powered chatbot built using the Google Gemini API. The chatbot generates intelligent and context-aware responses based on user input in real-time.
The goal of this project was to understand API integration, asynchronous JavaScript, and dynamic UI updates while building a modern AI-based web application.
🚀 Features
💬 Real-time AI responses
🔗 Integration with Gemini API
⚡ Fast and interactive UI
📱 Responsive design
🎨 Clean and minimal interface

🛠️ Technologies Used
HTML5
CSS3
JavaScript (Async/Await, Fetch API)

Google Gemini API
